# Tic Tac Toe :rocket:

This a terminal tic tac toe game in ruby 

--------

## Installation 
Clone using this command or fork the project
```sh
$ clone https://github.com/elio75006/morpion.git
``` 
```sh 
bundle instal
```
----------

## How to play
It's pretty simple, to watch engine play type: 
```sh
$ ruby app.rb
```
have fun!
